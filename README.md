# LinekedIn profile Clone
This project is a UI clone of a LinkedIn profile page, created as part of my internship task. The goal was to closely replicate the layout and design of a real LinkedIn profile while maintaining clean structure and responsiveness.
# Tech Stack
  - HTML
  - Tailwind CSS
# Features
  - LinkedIn-style profile layout
  - Responsive design for mobile and desktop
  - Clean and minimal UI
